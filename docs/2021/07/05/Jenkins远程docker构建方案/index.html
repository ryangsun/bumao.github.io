Jenkins远程docker构建方案

背景介绍

Jenkins，包括idea，有很多集成方案构建docker，但是因为网络结构、构建环境等客观原因，集成性的方案并不能满足需求。如：

  • Jenkins权限限制；
  • Jenkins已经是docker化部署，无法嵌套集成方案；
  • 构建环境不唯一，需要多个docker环境分别构建；
  • Jenkins端无私仓权限，无法把构建好的docker部署到指定位置。

这时，基于DinD的构建方案就非常方便了。
